# 去中心化身份(DID)系统前端

这是一个基于React的去中心化身份系统前端应用，用于与以太坊智能合约交互，实现DID的注册和查询功能。

## 功能特点

- 连接MetaMask钱包
- 使用测试钱包进行本地测试（无需安装MetaMask）
- 注册新的去中心化身份(DID)
- 查询地址对应的DID信息
- 响应式设计，适配移动端和桌面端

## 安装步骤

### 前提条件

- 安装Node.js (推荐v16.0.0或更高版本)
- 安装MetaMask浏览器扩展（如果使用真实钱包）
- 已部署DID智能合约
- 运行本地以太坊测试网络（如Hardhat Network或Ganache）

### 安装依赖

```bash
cd frontend
npm install
```

### 配置

在使用前，请确保更新`App.js`文件中的合约地址：

```javascript
// 合约地址 - 需要在部署后更新
const contractAddress = "0x5FbDB2315678afecb367f032d93F642f64180aa3";
```

将上面的地址替换为您实际部署的DID合约地址。您可以在运行部署脚本后在控制台输出中找到这个地址。

### 启动应用

```bash
npm start
```

应用将在开发模式下启动，访问 [http://localhost:3000](http://localhost:3000) 可以在浏览器中查看。

## 使用说明

1. **选择钱包类型**：
   - **真实钱包**：默认使用MetaMask钱包连接。
   - **测试钱包**：勾选"使用测试钱包"选项，可以选择预设的测试账户进行操作，无需安装MetaMask。

2. **连接钱包**：点击"连接MetaMask钱包"或"连接测试钱包"按钮，根据您的选择连接相应的钱包。

3. **注册DID**：在连接钱包后，输入您想要注册的DID值（例如：did:example:123），然后点击"注册DID"按钮。

4. **查询DID**：输入以太坊地址，点击"查询DID"按钮，即可查看该地址关联的DID信息。

## 与智能合约的交互

本应用使用ethers.js库与以太坊区块链进行交互。主要交互包括：

- `registerDID(did)`: 注册新的DID
- `getDID(address)`: 查询指定地址的DID信息

## 注意事项

- 使用真实钱包时，确保MetaMask连接到正确的网络（本地开发时通常是Hardhat Network或Ganache，地址为http://127.0.0.1:8545）
- 使用测试钱包时，确保本地测试网络（如Hardhat Network）正在运行
- 测试钱包已预设了3个Hardhat默认测试账户，可以直接使用
- 每个以太坊地址只能注册一个DID
- 交易需要支付少量gas费用（测试网络上通常有足够的测试ETH）

```bash
npm run build
```
